# 🍎 Лови яблоки! - версия 1.0

**Классическая аркадная игра** для лицейского проекта. Управляйте корзиной и собирайте падающие яблоки, чтобы набрать как можно больше очков!

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `←` (Стрелка влево) | Двигать корзину влево |
| `→` (Стрелка вправо) | Двигать корзину вправо |
