## 🚀 Как запустить

### Требования

- Установленный Python 3.1 или новее
- Библиотека `pygame` 2.6 или новее

### Установка и запуск

#### 1. **Склонируйте репозиторий:**
   ```cmd
   git clone https://github.com/Transistor-152/PygameSchoolProject1.git
   cd version_1/main_V1.py
   ```
#### 2. **Создайте виртуальное окружение (рекомендуется):**

  ##### **Windows (CMD)**
  ```cmd
  python -m venv venv
  venv\Scripts\activate
  ```
   
  ##### **macOS / Linux**
  ```bash
  python3 -m venv venv
  source venv/bin/activate
  ```

#### 3. **Установите зависимости:**
   ```cmd
   pip install -r requirements.txt
   ```
#### 4. **Запустите код:**
   ```cmd
   python main_V1.py
   ```
