## 🎮 Суть игры

С неба падают яблоки. Ваша задача — **поймать их в корзину**, перемещая её влево и вправо. За каждое пойманное яблоко начисляются очки. Если вы упустите 3 яблока — игра закончится.

> 🎯 **Цель игры:** набрать максимальное количество очков и побить собственный рекорд!

---

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `1` | Выбрать уровень легко |
| `2` | Выбрать уровень нормально |
| `3` | Выбрать уровень сложно |
| `←` (Стрелка влево) | Двигать корзину влево |
| `→` (Стрелка вправо) | Двигать корзину вправо |
| `R` | Начать новую игру (после проигрыша) |
| `ESC` | Выйти из игры |

## 🚀 Как запустить

### Требования

- Установленный Python 3.1 или новее
- Библиотека `pygame` 2.6 или новее

### Установка и запуск

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/ваш-username/lovi-yabloki.git
   cd v1/main.py

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
    Windows
    python -m venv venv
    venv\Scripts\activate

   macOS/Linux
   python3 -m venv venv
   source venv/bin/activate

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt

4. **Запустите код:**
   ```bash
   python main.py
