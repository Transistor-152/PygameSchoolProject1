# 🍎 Лови яблоки!

**Классическая аркадная игра**. Управляйте корзиной и собирайте падающие яблоки, чтобы набрать как можно больше очков!

## 🎮 Суть игры

С неба падают яблоки. Ваша задача — **поймать их в корзину**, перемещая её влево и вправо. За каждое пойманное яблоко начисляются очки.

![Пример игры](assets/GIF1.gif)

> 🎯 **Цель игры:** набрать максимальное количество очков и побить собственный рекорд!

---

## 🕹️ Управление

| Клавиша | Действие |
|---------|----------|
| `←` (Стрелка влево) | Двигать корзину влево |
| `→` (Стрелка вправо) | Двигать корзину вправо |
